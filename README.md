# 🏨 Hostel Ops – Complaint Management System

A full-stack Hostel Complaint Management System built using React, Node.js, Express, and MySQL.  
This system allows students to raise complaints and enables admins to manage and track them efficiently.

---

## 🚀 Features

- 🔐 JWT-based Authentication (Login/Register)
- 👤 Role-based Access Control (Student / Admin)
- 📝 Complaint Creation & Tracking
- 📊 Status Management (Pending, In-Progress, Resolved, Rejected)
- 🛡 Secure Environment Variable Configuration
- 🌐 RESTful API Architecture

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MySQL
- JWT (jsonwebtoken)
- bcryptjs

### Database
- MySQL 8.x
